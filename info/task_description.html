Get URLs in CJK communications.

Ref: List of allowed URL characters

Input: An string in non-Roman with URLs.

Output: An list contains all URLs in class string.

Example: 

checkio("分享兩個連結:"
        "死亡之握-http://goo.gl/abc5566"
        "馬耳卡茸-https://goo.gl/def7788") == ["http://goo.gl/abc5566", "https://goo.gl/def7788"]
checkio("各位同學, 這次報告要使用的server在sip://taiwan.tai"
        "pei/A___Aa,請大家在午夜十二點前交出 by 助教 >///<") == ["sip://taiwan.taipei/A___Aa"]
